## Architecture Notes
- generate.js: `generateEdgyBoyV2SlotScript()` wraps `generatePassingScriptForSlot()`
  - When disabled: calls baseline directly, identical output
  - When enabled: picks blueprint, generates N candidates, scores with harness, returns best
  - On all-fail: logs to logs/<date>/edgy-boy-v2-fail-log.json, returns baseline
- llm.js: `buildEdgyBanterPrompt(blueprint, ...)` builds blueprint-driven user prompt
- edgy-boy-harness.js: `scoreScript(script, cfg)` and `batchSummary(scripts, cfg)`
  - Option B rule: "yes??" / "yes?" = CONFUSION, not eager accept
  - Boy word count: reply.text + all boy messages
- edgy-boy-blueprints.js: 7 weighted blueprints, `pickBlueprint(mode, rng)`
